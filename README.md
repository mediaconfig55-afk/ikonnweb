# Klima Web Sitesi - Icon Projesi

Bu proje, bir klima tanıtım ve satış sitesi için geliştirilmiş modern bir web uygulamasıdır.

## Özellikler

- Modern ve şık kullanıcı arayüzü.
- Hızlı performans için Vite kullanımı.
- Duyarlı (responsive) tasarım.

## Kurulum ve Çalıştırma

Projeyi yerel makinenizde çalıştırmak için şu adımları izleyin:

1. Bağımlılıkları yükleyin:
   ```bash
   npm install
   ```

2. Geliştirme sunucusunu başlatın:
   ```bash
   npm run dev
   ```

3. Üretim sürümünü oluşturun:
   ```bash
   npm run build
   ```
